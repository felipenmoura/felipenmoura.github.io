Syntax Highlight

You can also use an addon for syntax highlight of source codes when you need it.
Você também pode usar o addon para syntax highlight de seus códigos fonte quando precisa. 
<?php
    function empty_dir($dir) {
        $iterator = new RII(new RDI($dir),
                            RII::CHILD_FIRST);
        foreach ($iterator as $path) {
          if ($path->isDir()) {
             rmdir($path->__toString());
          } else {
             unlink($path->__toString());
          }
        }
        // rmdir($dir);
    }
?>
