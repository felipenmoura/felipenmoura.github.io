Um pouco de História

  • Em 2007 Webkit anuncia que está iniciando o desenvolvimento de animações em CSS, bem como os efeitos transform.

  • Apenas em 2009 a W3C iniciou os trabalhos iniciais de tais implementações para padronização

  • Em 2011, Firefox 5 passou a incluir animações em CSS
Hoje, animações em CSS3 são implementadas até mesmo no iTunes.
Em 2011, a Microsoft anuncia que Windows 8 será baseado em HTML5, Javascript e CSS para seus aplicativos.
