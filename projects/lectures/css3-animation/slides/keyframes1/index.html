Keyframes




Permitem a criação de animações com muito maior controle
@-vendor-keyframes nome-animacao{ 0%{ propriedade: valor; } 50%{ propriedade: valor; } 100%{ propriedade: valor; } }



Por exemplo:
@-webkit-keyframes anim1{ from{ background-color: green; } to{ background-color: blue; } }
@-webkit-keyframes anim1{ from{ background-color: green; } to{ background-color: blue; } }
