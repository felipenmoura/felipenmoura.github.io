Keyframes






Agora que ja criamos uma animação,
Como a usamos?


Especificamos o nome da animação

-webkit-animation-name: anim1;


Especificamos a duração

-webkit-animation-duration: 4s;

Usamos s para segundos
ou ms para milesegundos


O número de vezes a executar

-webkit-animation-iteration-count: infinite;

Podemos usar um número, ou infinite


Então, definimos a direção

-webkit-animation-direction: alternate;

Os valores são:
  alternate: vai, e volta
  normal: apenas ida


Podemos definir um efeito de suavização

-webkit-animation-timing-function: ease-out;

Com algum dos valores são:
  ease,
  ease-out,
  ease-in,
  ease-in-out,
  linear,
  cubic-bezier(x1, y1, x2, y2)


Propriedades padrão

-webkit-animation-fill-mode: forwards;

Deixa o elmento em um formato logo no início da animação
Aceita: forwards, backwards, both, none


Tempo de delay

-webkit-animation-delay: 2s;

Um valor em segundos(s) ou milesegundos(ms)


Resumindo...para cada vendor usamos:

-webkit-animation-name: bounce; -webkit-animation-duration: 4s; -webkit-animation-iteration-count: 10; -webkit-animation-direction: alternate; -webkit-animation-timing-function: ease-out; -webkit-animation-fill-mode: forwards; -webkit-animation-delay: 2s;


Mas não se preocupem, não precisamos usar aquilo tudo!
.animated{ -webkit-animation: anim1 2s infinite alternate; -moz-animation: anim1 2s infinite alternate; -ms-animation: anim1 2s infinite alternate; -o-animation: anim1 2s infinite alternate; animation: anim1 2s infinite alternate; }
